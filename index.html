dependency-highlighter by yihuang

a simple tool to highlight cabal package dependency graph.

Authors

yihuang (yi.codeplayer@gmail.com)

Contact

yihuang (yi.codeplayer@gmail.com)

How to use

Click a node, then all the nodes directly connected with it will be highlighted. Open pkgs.html with any modern browser to see example. Feature request is welcome.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/yihuang/dependency-highlighter

get the source code on GitHub : yihuang/dependency-highlighter
